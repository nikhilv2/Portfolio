IoT and Electronics Based Projects Development 

1. Smart Programmable Logic Controller

The Project included:

  • Generating ladder logic diagrams using OpenPLC Editor for different instructions
  • Structure Text (.st) files are generated for each
  • A PLC Runtime was generated using matiec compiler
  • Using Shunya OS as the developing platform, the output was tested and debugged on Raspberry Pi 4
  • Implemented the Communication modules

Technologies Used:

  • Languages - C, C++
  • Tools - OpenPLC Editor, Shunya OS
  • Board - Raspberry Pi 4
  • Communication Protocol - Modbus

2. Non-invasive Cuffless Blood Pressure Monitoring using PPG Signal

The Project included:

  • A system was proposed to measure blood pressure using PPG signal data obtained from a finger.
  • An electronic circuit is implemented to calibrate and monitor the PPG signal.
  • Using AI techniques, the value of blood pressure could be estimated.

Technologies Used:

  • Languages - Assembly, Python
  • Tools - Matlab
  • Components
    • PPG Sensor
    • Instrumentation Amplifier AD620
    • Op Amp 741
    • Microcontroller - ATmega 328
    • LCD Display

3. Home Automation using IoT, Blynk app and AWS cloud

The Project included:

  • A system was designed to control home appliances to utilize power efficiently from any location at any time
  • The home appliances were controlled using a relay module and linked via WIFI using NodeMCU ESP8266 chip.
  • The output data could be monitored through AWS cloud server and the devices could be controlled by Blynk application

Technologies Used:

  • Languages - C, C++, PHP, HTMl
  • Tools - Arduino, AWS Cloud
  • Board - ESP8266 (NodeMCU v3)
